# bravo11-friend-finder-backend

## How to Run
1. Download MySQL
2. Run the following command, templateDB.sql can be found in /template-data:
    - ```mysql -u username -p```
    - ```CREATE DATABASE team_management;```
    - ```exit;```
    - ```mysql -u usesrname -p team_management < templateDB.sql```
3. node .\server\
